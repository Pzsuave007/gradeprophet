<analysis>**original_problem_statement:**
The user wants to build a web app named GradeProphet to help them decide which sports cards to send for professional grading by PSA.

**PRODUCT REQUIREMENTS:**
1.  **Core Functionality:** Upload photos of sports cards (front, back, and optional corners), and have an AI analyze them based on PSA's grading standards (centering, corners, surface, edges) to predict a potential grade.
2.  **AI Integration:** Use OpenAI's Vision model.
3.  **PSA 10 Reference Library:** Ability to upload a reference image of a PSA 10 graded version of the same card. The system should automatically read the text from the PSA slab's label (including the year) and store it in a reusable library.
4.  **Analysis History & Feedback Loop:** Keep a history of all analyses and allow the user to input the actual grade received from PSA to create a feedback loop for the AI.
5.  **Card Age Consideration:** The AI must consider the age of the card in its evaluation, applying more lenient standards for vintage cards. This should be automated by reading the year from the PSA 10 reference or by AI estimation if no reference is provided.
6.  **eBay Integration (New):**
    *   Allow the user to paste an eBay listing URL (both full and short  links).
    *   The system should automatically import all images from the listing.
    *   Provide a UI for the user to assign the imported images as Front, Back, or specific Corners.
    *   If the listing doesn't include corner photos, the system should auto-generate corner/edge crops from the best front image.
7.  **UI/UX:** A professional dark theme, mobile-friendly, with easy access to all options. Remove estimated market value from the results.
